# 🚗 Nostalgic Cars

Um projeto desenvolvido para praticar HTML5 durante meus estudos de desenvolvimento web.

## 📖 Sobre o projeto

O Nostalgic Cars é um site dedicado a carros icônicos de filmes e jogos. O objetivo foi colocar em prática conceitos básicos de HTML, criando uma página organizada e sem utilizar frameworks.

## ✨ Funcionalidades

- Página inicial
- Menu de navegação
- Seção de carros de filmes
- Seção de carros de jogos
- Formulário de contato
- Estrutura semântica com HTML5

## 🛠️ Tecnologias utilizadas

- HTML5

## 🎯 Objetivo

Este projeto faz parte da minha jornada como desenvolvedor. Ele foi criado exclusivamente para praticar os conhecimentos adquiridos durante meus estudos.
